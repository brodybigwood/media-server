# Media Server

## Setup

For authentication, set the environment variable MEDIA_API_KEY to your desired key

## Usage

- run the server by going into the server folder and running "dotnet run"

- visit the website or make http api requests using port 5000

- the current method to add files is to symlink or paste the folder containing the media into the main directory, change "files" inside addfolder.py to the name of the folder, and run addfolder.py

## Requirements

- .NET SDK 10.0 or later

- see requirements.txt for python requirements
